## Composantes du site web

### Navigation (Header)

Le site comprend une barre de navigation fixe en haut de chaque page avec :
- **Logo "Claire"** : Un lien cliquable qui ramène toujours à la page d'accueil
- **Accueil** : Lien vers la page d'accueil
- **Mon parcours** : Lien vers la page présentant mes œuvres artistiques
- **Inspirations** : Lien vers la page des artistes qui m'inspirent
- **S'inscrire** : Bouton d'appel à l'action menant à la page de contact

---

### Page d'Accueil (index.html)

La page d'accueil présente une introduction générale au site web et à mon parcours en tant qu'élève de 2de4:

- **Section d'introduction** : Présentation personnelle et contexte du projet dans le cadre du cours de SNT
- **Un peu sur moi** : Trois cartes interactives présentant mes passe-temps principaux (Basketball, Lecture, Danse). Au survol de la souris, les cartes révèlent des descriptions détaillées de chaque activité
- **Ma musique** : Section présentant mes styles musicaux préférés (pop, jazz, rock, classique) avec des lecteurs Spotify intégrés pour trois artistes favoris (Frank Sinatra, Ado, ENHYPEN), permettant d'écouter leur musique directement sur le site

### Mon Parcours

Cette page présente mon parcours artistique et personnel, avec une attention particulière portée sur mes réalisations créatives récentes. Elle comprend :

- **Introduction** : Description de mon implication scolaire (déléguée de classe, représentante au CVL) et de ma passion pour l'art
- **Portfolio artistique** : Trois œuvres récentes présentées avec image à droite et description détaillée à gauche :
  - *Metamorphosis* (2025) : Autoportrait au crayon mêlant traits humains et animaux
  - *REM* (2023) : Illustration aquarelle et encre sur le nouveau train REM
  - *Watercolor Bear* (2022) : Études à l'aquarelle explorant les techniques de dilution et de contraste
